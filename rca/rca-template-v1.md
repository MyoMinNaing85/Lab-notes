# RCA Template v1
(Root Cause Analysis – Life & Practice)

## 1. Phenomenon (ဖြစ်ပေါ်လာတဲ့ အရာ)
- ဘာဖြစ်တာလဲ
- ဘယ်အချိန် / ဘယ်နေရာ
- ကိုယ့်အတွင်းမှာ ဘာခံစားရလဲ

## 2. Surface Cause (မြင်ရတဲ့ အကြောင်းရင်း)
- အပြင်ပန်း အကြောင်းရင်း
- လူ / အခြေအနေ / ပတ်ဝန်းကျင်

## 3. Deeper Cause (အတွင်းကျသော အကြောင်းရင်း)
- အလေ့အကျင့်
- အယူအဆ
- အလိုဆန္ဒ (tanha)
- မသိမှု (avijja)

## 4. Buddha Concept Mapping
- အနစ္စ (Impermanence)
- ဒုက္ခ (Unsatisfactoriness)
- အနတ္တ (Non-self)
- ပဋိစ္စသမုပပါဒ (Dependent Origination)

## 5. Pattern Recognition
- ဒီလိုအခြေအနေ မတိုင်ခင်လည်း ဖြစ်ဖူးလား
- Pattern ပြန်ပေါ်နေတာလား

## 6. Skillful Response (လက်တွေ့ အသုံးချ)
- မလုပ်သင့်တာ
- လုပ်သင့်တာ
- လက်တွေ့ လေ့ကျင့်ချက် (small action)

## 7. Reflection (နောင်အတွက် သင်ခန်းစာ)
- ဒီအဖြစ်အပျက်က ဘာသင်ပေးလဲ
- နောင်မျိုးဆက်အတွက် ဘာကျန်ထားမလဲ
